# Módulos

## O que é um módulo

- Exemplo seria nosso corpo, precisamos de determinadas coisas para sobreviver ao dia.
> Ex: bebibas, comida, doce, etc...

- Estes são os módulos, são bibliotecas que necessitamos usar em nossa aplicação.
- Para usar os módulos, usamos comandos de importação.

## Importando um módulo

- Exemplos:
> `import bebida`
> `import doce`

- Para importar apenas um especifico, invertemos:
> `from doce import pudim`

### Importando o módulo math

- Com a lib math, podemos usar ceil, floor e trunc para arredondar os numeros, existem também a pow, sqrt, factorial, etc

- Importando tudo de math:

```py
import math
```

- Para usar apenas sqrt para calcular raiz quadrada do módulo math:

```py
from math import sqrt
```
